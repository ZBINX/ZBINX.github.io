Virtualbox Guest Additions install / guide

November 25, 2021 2-minute read

What is VirtualBox guest additions?

Guest Additions is simply just an add-on for your VirtualBox application so you can fully utilize vbox to its full potential. Some people think that once you install virtualbox and the OS of your choice, that you are then good to go. If you do not install guest additions on top of your new OS vbox, then you might not have full resolution, dual monitor support, full screen, copy / paste etc….

How to install guest additions ( linux )

  • 1st boot up your virtualbox environment and OS.

  • Navigate to the VBOX menu ( outside of the envinronment) and click devices > optical drives > choose / create disk image.

  • click the latest VBoxGuestAdditions.iso you see from the list and then click choose.

  • If you see a question prompt about “Force Umount” , then click Force Unmount

  • This should force attach your guest addition iso to your virtual environment.

  • At this point, your guest additions iso add on is now connected and ready to be installed inside of your vbox.

  • open up a terminal and navigate to cd ~/media/username/Vboxdirectory . The guest additions iso should mount at this location on your linux vbox, if it does not then you need to find where it is mounted and then run the terminal from that location.

  • run the ls command to list all the files in this folder. We are looking for the install file labeled VboxLinuxAdditions.run

  • Now install the guest additions by issuing the following command in your terminal:

    • sudo ./VBoxLinuxAdditions.run

  • Once installation is complete you need to reboot for the changes to take effect.

  • Enjoy!
