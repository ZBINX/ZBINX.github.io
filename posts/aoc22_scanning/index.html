What is Scanning?

Scanning is a set of procedures for identifying live hosts, ports and services, discovering the OS of target system & identifying vulnerabilities & threats in the network. Scanning reveals parts of the attack surface for attackers and allows launching targeted attacks to exploit the system.

Scanning Types

Scanning is classified as either active or passive based on the intrusiveness to gathering info about a target system or network.

Passive Scanning : scanning network without directly interacting with the target. Usually carried out through packet capture & analysis with tools like wireshark; however this technique only provides basic asset info like OS version, network protocols etc…

Active Scanning : Sending packets or queries directly to specific assets to retrieve more detailed information.

Scanning Techniques

Network Scanning

Network scanning helps to discover a map of the network. Including any live computers or hosts, open ports, IP addresses and services running.

Port Scanning

As per wikipedia, “In computer networking, a port is a number assigned to uniquely identify a connection endpoint and to direct data to a specific service. A port is a logical construct that identifies a specific process or a type of network service.”

Port scanning is a conventional method to examine open ports in a network capable of receiving and sending data. Port scanning results fall into the following 3 categories:

  • Closed Ports: host is not listening to the specific port.
  • Open Ports: The host actively accepts a connection on the specific port.
  • Filtered Ports: This indicates that the port is open; however, the host is not accepting connections or accepting connections as per certain criteria like specific source IP address.

Vulnerability Scanning

The vuln scanning proacively IDs the networks vulns in an automated way that helps determine whether the system may be threatened or exploited. Pentesters widely use tools such as Nessus and Acunetix to identify loopholes in a system.

Scanning Tools

Network Mapper (NMAP)

Nmap can port scan, discover network protocols, id running services, and detect OS on live hosts.

  • TCP SYN Scan: scanning ports on hosts without completing TCP 3 way handshake. Little more stealthier.

    Usage: nmap -sS ipaddress

  • Ping Scan : allows scanning live hosts without checking for services.

    Usage: nmap -sn ipaddress

  • OS scan : allows scanning of the type of OS running.

    Usage: nmap -O ipaddress

  • Detecting Services : get a list of running services

    Usage: nmap -sV ipaddress

Nikto

Nikto is open-source software that allows scanning websites for vulns. It enables looking for subdomains, outdates servers, debug msgs etc on website.

Usage: nikto -host ipaddress
